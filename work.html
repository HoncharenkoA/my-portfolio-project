Weather app

The project provides information for any specific city about the current weather, humidity, wind, pressure. Added hourly and weekly weather forecasts. The application presents real-time weather data and forecast for the next 5 days for any city in the world. Also you can see current weather for 5 countries proposed by me. The app is responsive and developed by Visual Studio. I also became skilled in Bootstrap, API's, real-life development workflow, hosting, and advanced JavaScript.

Built with HTML, CSS and Java Script.
